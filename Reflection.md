# Reflection Questions

## 1. How did AI tools support your Markdown documentation process?

AI tools helped generate a professional structure for the README and R Markdown documentation files. They also assisted with formatting, organizing sections, and improving clarity and consistency in the Markdown syntax.

---

## 2. What strengths and weaknesses did you notice in AI-generated documentation?

One strength of AI-generated documentation is speed and organization. The AI was able to quickly generate clean Markdown formatting and professional section layouts. However, some outputs required manual review because certain explanations were too generic or needed refinement to better match the project requirements.

---

## 3. How did you verify and improve the AI-generated outputs?

I verified the outputs by previewing the Markdown formatting in GitHub and reviewing the structure for readability and correctness. I also manually edited sections, adjusted formatting, and ensured that headers, code blocks, and lists rendered properly before finalizing the files.
