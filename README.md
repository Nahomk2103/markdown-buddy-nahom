# Student Performance Analysis

## Overview
This project demonstrates a basic R data analysis workflow for evaluating student academic performance using synthetic datasets. The project focuses on cleaning data, generating summary statistics, and visualizing trends related to student grades and attendance.

The repository was created as part of the BDA400 coursework on Markdown documentation and AI-assisted project formatting.

---

## Project Structure

- `README.md` — Main project documentation
- `ScriptDocumentation.Rmd` — R Markdown documentation for the analysis script
- `student_analysis.R` — Sample R analysis script
- `Reflection.md` — Reflection on AI-assisted documentation workflow

---

## Features

- Data cleaning using R
- Summary statistics generation
- Visualization of student performance trends
- Markdown and RMarkdown documentation
- GitHub repository organization
